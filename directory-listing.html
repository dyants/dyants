Directory Listing

  1. Home
  2. Directory Listing

Information

  • Category: Penetration Testing
  • Target: Web Hosting
  • Project date: 01 September, 2025
  • Project URL: www.anonim.com

Summary

Saat melakukan pemeriksaan keamanan terhadap domain utama dan lebih dari 50 subdomain-nya saya berhasil menemukan kerentanan Directory Listing (atau Directory Traversal/Listing) pada sebuah subdomain, yaitu sub.domain.com.

Kerentanan ini secara tidak sengaja mengekspos daftar isi direktori server yang terlihat seperti repositori paket perangkat lunak, khususnya paket .rpm yang terkait dengan AlmaLinux dan GPG Keys (misalnya, almalinux-gpg-key-latest-10.x86_64.rpm)

Seperti yang terlihat pada gambar, direktori tersebut memungkinkan pengunduhan dan pengecekan file secara langsung oleh publil. Meskipun celah ini telah dilaporkan dan diperbaiki oleh Tim Developer, temuan ini menunjukan adanya konfigurasi web server yang tidak aman sebelumnya.

Ringkasan Temuan

Tingkat Keparahan

Kerentanan ini dikategorikan sebagai Medium Severity Karena memungkinkan pihak luar untuk membaca isi file yang berada dalam direktori tersebut.

Deskripsi Kerentanan

Kerentanan ini memungkinkan akses langsung terhadap file dalam direktori tanpa autentikasi. Hal ini berpotensi mengungkap informasi melalui daftar file yang dapat diakses publik.

Dampak

Kerentanan ini dapat menyebabkan kebocoran informasi, seperti file log, konfigurasi, atau data sensitif lain yang tersimpan di direktory tersebut. informasi dapat dimanfaatkan untuk serangan lanjutan.

Mitigasi

Melakukan konfigurasi ulang web server untuk menonaktifkan directori listing (misal melalui .htaccess, konfigurasi Nginx, atau Apache). Selain itu, dilakukan audit keamanan pada seluruh direktori untuk memastikan tidak ada folder lain yang dapat diakses secara publik.
